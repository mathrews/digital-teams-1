# Objetos:

- Formas de agrupar tipos de informações.

- Possui propriedades.

- Funciona entro {}.

## sintaxe:

~~~JS exemplo:
let aluno = {
    nome: 'Mateus',
    idade: 15,
    genero: 'm'
    gostosMusicais:[
        'metal',
        'forró'
    ]
}
~~~

## array de objetos:

~~~JS exemplo
const teams = [
    {
        nome: 'nome 1',
        qtd: 10,
        participantes: ['mateus', 'melissa']
    },
    {
        nome: 'nome 2',
        qtd: 10,
        participantes: ['luis', 'marjorie', 'lorena']
    }
];
~~~

- Parâmetro é uma variável que só faz sentido dentro da função
